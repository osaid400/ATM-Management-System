
# ATM Management System (Python)

A beginner-friendly ATM Management System built using Python. This project simulates the basic functionality of an ATM using functions, loops, lists, and dictionaries.

## Features

- User Login with Account Number and PIN
- Check Account Balance
- Deposit Money
- Withdraw Money
- Change PIN
- Logout
- Input Validation
- Insufficient Balance Check
- 4-Digit PIN Validation

## Technologies Used

- Python 3

1. Clone the repository.

```bash

```

2. Open the project folder.

3. Run the Python file.

```bash
python "ATM Management System.py"
```
